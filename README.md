# CodeAlpha_-Exploratory-Data-Analysis-EDA-
This project focuses on performing Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python. The main objective is to understand the structure of the dataset, identify trends, patterns, and insights, and visualize important information related to Netflix content.


## Questions Answered

1. Distribution of Movies vs TV Shows  
2. Top 10 content-producing countries  
3. Yearly trend of content added  
4. Most common ratings  
5. Movie duration distribution (normal vs skewed)  
6. Top directors, cast members, and genres  
7. Seasonal patterns of content addition
